Working with colors and animations

See the full project

For the first part of the assignment, I chose two bright accent colors, bright salmon and light blue. I also used gray and cream as my neutral colors.

For the second part of the assignment, I created a dark theme for the landing page. I also used the neutral colors in different ways to create a cohesive and readable design. For example, I used cream as the background color and light gray as the text color to create a light mode for the landing page. When switching to dark mode, I used dark gray as the background color and light cream as the text color to ensure readability.

Finally, for the last part of the assignment, I added animations to the landing page design to create a more dynamic user experience.
